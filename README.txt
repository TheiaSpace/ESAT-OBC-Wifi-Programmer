ESAT OBC Wifi Programmer version 1.0.0.

Use the ESAT OBC board as a programming board for the ESAT Wifi board.

This software runs in the ESAT OBC board.  Use it with Arduino 1.8.0
or newer together with the Arduino core for MSP430-based ESAT boards
(Theia Space's ESAT Boards (MSP430)) 3.0.0 or a newer 3.x.y version).
Use ESAT-OBC as the target board.

See the example program
(examples/ESAT-OBC-Wifi-Programmer/ESAT-OBC-Wifi-Programmer.ino).
